# sajureport

## Setup

1) Install
```
npm install
```

2) Env
- Copy `.env.example` to `.env.local` and fill in values.
- 배포 플랫폼에서도 동일한 환경변수를 등록해야 합니다.

3) Run
```
npm run dev
```

## Environment Variables

```
GEMINI_API_KEY=
ADMIN_SECRET=
NEXT_PUBLIC_ADMIN_SECRET=
NEXT_PUBLIC_SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
```
